##### Viral and fungal pathogens associated with Pneumolaelaps niutirani (Acari: Laelapidae),
##### a mite found in diseased nests of Vespula wasps (Felden et al. 2019)
##### Supplementary information
##########################################################################################################

The following folder contains all supporting information for the article 10.1007/s00040-019-00730-y.
Clean RNA-seq reads can be accessed from the NCBI SRA server, BioProject ID PRJNA576756 (https://www.ncbi.nlm.nih.gov/sra/PRJNA576756). 

01-data contains:
	
	blastx_fungi.outfmt6
		DIAMOND blast output for fungal transcripts
	blastx_viruses.outfmt6
		DIAMOND blast output for viral transcripts
	gene_count_matrix.csv
		Vespula vulgaris transcript counts (used for library size)
	genes.TMM.EXPR.matrix
		output of Trinity containing the TMM-normalised counts for non-Vespula transcripts (includes viral and fungal transcripts)
	LC_transcripts_edited.csv
		Edited version of LC_transcripts.csv file produced in R (i.e. with added viral families and host)
	protein_result_fungi.csv
		Formatted output of fungal protein hits from NCBI batch entrez (https://www.ncbi.nlm.nih.gov/sites/batchentrez)
	protein_result_viruses.csv
		Formatted output of viral protein hits from NCBI batch entrez (https://www.ncbi.nlm.nih.gov/sites/batchentrez)
	qPCR_taqman_array_KBV_Aspergillus.csv
		RT-qPCR data for Aspergillus and KBV

02-scripts contains:

	mites_wasps.R
		R script including the rest of the analysis, the two variables DataDir and FigDir have to be changed according to the user's file system

03-figures is an empty directory where figures generated by the R script are saved
